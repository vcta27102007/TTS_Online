# Tuần 1 - User / Task / Project

Project tuần 1 tập trung vào:
- đọc yêu cầu domain và ghi chú nghiệp vụ
- xác định entity, field, enum
- viết class OOP cho User / Project / Task
- viết logic add / update / delete Task
- gán Task cho User
- xử lý lỗi logic cơ bản
- test console và ghi log

## Cách chạy
Chạy class `com.example.demo.App`

## Kết quả chính
- docs phân tích domain trong thư mục `docs`
- code OOP trong `src/main/java/com/example/demo`
- demo flow console trong `App.java`
